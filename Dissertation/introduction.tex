% В первой главе представлен анализ существующих стандартов беспроводных широкополосных сетей. Рассмотрены архитектура, логическая и физическая структура сетей для стандарта IEEE 802.11, а также сотовых сетей LTE и 5G. Представлен обзор известных моделей распространения радиосигналов. С их помощью рассчитываются такие параметры, как радиус телекоммуникационной связи между базовыми станциями и радиус телекоммуникационного покрытия транспортной магистрали, необходимые для формулировки и решения задачи оптимального размещения базовых станций беспроводной сети. 

% Во второй главе сформулирована задача целочисленного линейного программирования (ЦЛП) по критерию максимальной области телекоммуникационного покрытия протяженной транспортной магистрали с ограничением на суммарную стоимость аппаратно-программных средств сети. Достоинством сформулированной модели в виде ЦЛП является возможность ее численного решения с использованием различных существующих программных комплексов. Однако в рамках такой задачи не рассматривается важное при проектировании топологии сети ограничение на межконцевую задержку передачи пакетов по сети. Для учета этого ограничения сформулирована комбинаторная модель задачи размещения базовых станций и разработан новый алгоритм специального типа (ветвей и границ) для ее решения. Для оценки межконцевых задержек построена аналитическая модель многофазной сети массового обслуживания с кросс-трафиком, с экспоненциальными распределениями времени между поступлениями пакетов и времени обслуживания на фазах. Разработана новая итерационная процедура нахождения последовательности лучших решений для задачи размещения базовых станций в рамках комплексного проектирования беспроводных широкополосных сетей для телекоммуникационного покрытия протяженных объектов.

% В третьей главе сформулирована и решена задача оптимального размещения базовых станций для обслуживания заданного множества рассредоточенных объектов в виде задачи частично целочисленного линейного программирования (ЧЦЛП). Если во второй главе рассмотрена задача максимального телекоммуникационного покрытия, в третьей главе рассматривается постановка задачи минимизации суммарной стоимости, где необходимым условием является обеспечения телекоммуникационной связью всех заданных стационарных объектов.

% В четвертой главе представлены численные примеры решения задач оптимального размещения. Приведено описания разработанного программного комплекса, а также сравнительный анализ предложенного комбинаторного алгоритма с результатом решения сформулированной задачи ЦЛП с использованием программных пакетов дискретной оптимизации. Сравнение результатов проводилось по количеству пройденных узлов в ходе поиска оптимального решения. Разработанный алгоритм для комбинаторной модели показывает свою эффективность при увеличении размерности задачи.
